Jak to cist

Index je lokalni rozhodovaci pomucka. Nerika "let/nelet", ale tridi den: vitr, termicky potencial, dostupovy proxy odhad, oblacnost, srazky a bourkovou poznamku. Bourky nejsou automaticky stop, jen vyrazna rizikova vrstva.
